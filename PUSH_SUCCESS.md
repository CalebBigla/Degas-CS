## Clone on Another Machine

To work on this project from another computer:

```bash
git clone https://github.com/CalebBigla/Degas-CS.git
cd Degas-CS
npm install
```

Then setup:
1. Copy `.env.example` to `.env` in backend and frontend
2. Configure environment variables
3. Run `npm start` in backend
4. Run `npm run dev` in frontend
